# AI Управление OpenCV

Веб-приложение для обработки изображений с использованием OpenCV и искусственного интеллекта. Приложение пон��мает команды на русском языке и преобразует их в операции обработки изображений.

## 🎯 Возможности

- **Загрузка изображений** - поддержка различных форматов (PNG, JPG, и др.)
- **Обработка по командам на русском языке** - пишите команды естественным языком
- **Быстрые кнопки** - предустановленные действия для быстрого доступа
- **LLM поддержка** - использование локальной LLM модели для понимания сложных команд
- **Локальный парсинг** - быстрая обработка стандартных команд без LLM

## 🔧 Поддерживаемые операции

1. **Поворот** - `{"action": "rotate", "angle": 90}` (90, 180, 270°)
2. **Масштабирование** - `{"action": "resize", "scale": 0.5}`
3. **Чёрно-белое** - `{"action": "grayscale"}`
4. **Извлечение канала** - `{"action": "extract_channel", "channel": "R"}` (R, G, B)
5. **Размытие** - `{"action": "blur", "kernel": 15}` (нечётное число)

1. Клонируйте репозиторий:
```bash

cd Computer_Vision
```

2. Установите зависимости:
```bash
pip install opencv-python gradio numpy openai
```

3. **Важно**: убедитесь, что у вас запущена локальная LLM модель на адресе `http://localhost:1234/v1` (например, LM Studio)

## 💻 Использование

Запустите приложение:
```bash
python main.py
```

Откройте браузер и перейдите по ссылке, которая появится в консоли (обычно `http://127.0.0.1:7860`).

## 🏗️ Архитектура

### Основные функции

- **`parse_command_locally()`** - локальный парсинг команд (быстро, для стандартных команд)
- **`parse_command_with_llm()`** - обработка команд через LLM (для сложных команд)
- **`apply_command()`** - применение операции к изображению
- **`process_text_instruction()`** - основная функция обработки текстовых команд
- **`process_quick_action()`** - обработка быстрых кнопок

### Структура данных команды

Все команды представляют собой JSON-объекты:
```json
{
  "action": "название_операции",
  "параметр1": "значение1",
  "параметр2": "значение2"
}
```

## 🔌 Конфигурация LLM

По умолчанию приложение использует:
- **Base URL**: `http://localhost:1234/v1`
- **API Key**: `lm-studio`
- **Model**: `local-model`

Для изменения конфигурации отредактируйте эти строки в `main.py`:
```python
client = OpenAI(base_url="http://localhost:1234/v1", api_key="lm-studio")
```

## 📝 Примечания

- Локальный парсинг команд работает очень быстро и поддерживает стандартные операции
- Для более сложных команд автоматически используется LLM
- Все изображения автоматически конвертируются в формат BGR/RGB для корректной обработки
- Ядро размытия автоматически делается нечётным (добавляется 1, если чётное)
- Масштаб изображения имеет минимальное ограничение 0.05
